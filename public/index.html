Gtypist + record

WHAT IS?

gnu-typist + 練習記録パッチ。

動作

練習ステージをひとつ終了したらその記録を ~/.gtypist に吐き出すよう、 gtypist を変更。

gtypist_swing.rb あるいは gtypist_tk.rb は定期的に ~/.gtypist を見て、 その更新状況をウィンドウ上に表示する。

gtypist.sh は記録を残すように改造した gtypist と共に viewer を立ち上げ、 gtypist の終了とともに viewer を終わらす。

INSTALL on LINUX

$ tar xfz gtypist-2.9.2.tar.gz
$ cd gtypist-2.9.2
$ ./configure --prefix=/edu
$ patch -p1 <../patch
$ make
$ make install

INSTALL on OSX

$ tar xfz gtypist-2.9.2.tar.gz
$ cd gtypist-2.9.2
$ patch -p1 <../patch.osx
$ ./configure
$ patch -p1 <../patch
$ make
$ make install
